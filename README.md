# EXPLICACION DE LA APLICACION
Esta aplicación móvil permite gestionar tareas de manera sencilla utilizando **Kotlin** y **Jetpack Compose**.  
El usuario puede:
- Crear nuevas tareas con un título y descripción.  
- Ver una lista completa de las tareas registradas.  
- Editar una tarea mediante un diálogo emergente.  
- Visualizar los detalles de una tarea al seleccionarla.  
- Marcar tareas como completadas.  
- Eliminar todas las tareas completadas.

La aplicación utiliza un ViewModel para el manejo del estado y sigue una arquitectura organizada que facilita la escalabilidad y mantenibilidad del proyecto.
